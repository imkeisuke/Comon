各種圧縮フォーマットを統一的なインタフェースで扱う

## Description
圧縮フォーマットにはいくつかの種類があるが、それらの圧縮ツールに用いられるインタフェースは異なる点がある。そこで、Comonを用いることで統一的なインタフェースを提供する。これにより、各圧縮フォーマットを容易に扱うことができる

## Usage
```sh
Comon [OPTIONS] <ARGUMENTS...>
OPTIONS
 -h, --help   Print this help message
ARGUMENTS
  
```
